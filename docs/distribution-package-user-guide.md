# 配布版 利用開始ガイド案

## 1. このガイドの対象
このガイドは、配布された ZIP を受け取って案件情報System を使い始める利用者向けの手順です。
PowerShell や特別な設定作業は不要です。

## 2. 利用者側の手順
1. 配布された `案件情報System.zip` を、デスクトップなど分かりやすい場所に保存してください
2. ZIP を右クリックして「すべて展開」を選んでください
3. 展開先は毎回同じ場所を使うことを推奨します
4. 例として、ドキュメント配下など分かりやすい固定場所を使ってください
5. 展開後のフォルダ名が `案件情報System` であることを確認してください
6. `案件情報System` フォルダを開いてください
7. フォルダ内の `案件情報System.exe` をダブルクリックしてください
8. `案件情報System.exe` を起動すると、同じフォルダ内の Kernel を基準に起動します
9. 以後は、展開したそのフォルダから使ってください

## 3. 利用者側の禁止事項
- ZIP の中身を一部だけ取り出さない
- ZIP を解凍せずに直接実行しない
- `Addins` フォルダを削除しない
- `雛形` フォルダを削除しない
- `logs` フォルダを削除しない
- `案件情報System_Kernel.xlsx` と `案件情報System_Base.xlsx` のファイル名を変更しない
- ネットワーク上で直接起動しない
- 古いフォルダに新しいファイルを上書きしない

## 4. 更新時の注意
新しい版を受け取った場合は、今使っているフォルダへ上書きせず、別フォルダとして新しく展開してください。
展開後のフォルダ名は `案件情報System` を使う前提です。
切り替え手順が必要な場合は、配布元からの案内に従ってください。

## 5. 配布物に含まれるもの
配布物には、次が含まれます。

- `案件情報System.exe`
- `案件情報System_Kernel.xlsx`
- `案件情報System_Base.xlsx`
- `利用開始ガイド.pdf`
- `Addins` フォルダ
- `雛形` フォルダおよびその中身一式
- `logs` フォルダ

`利用開始ガイド.pdf` は現状では仮案ですが、当面はこの形で同梱されます。

## 6. 困ったとき
- 起動できない
- フォルダが足りない
- 展開方法が分からない

このような場合は、ZIP を加工せず、そのまま配布元へ連絡してください。
