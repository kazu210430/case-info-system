# 配布版 利用者ガイド

## 1. このガイドの対象
- このガイドは、配布された `案件情報System.zip` を受け取った利用者向けの手順です
- 初回セットアップは `初回セットアップ.bat` を使います
- PowerShell は使いません
- `.vsto` を直接開く必要はありません

## 2. 使い始める手順
- `案件情報System.zip` を分かりやすい場所に保存します
- ZIP を右クリックして「すべて展開」を選びます
- 展開後のフォルダ名が `案件情報System` であることを確認します
- 展開した `案件情報System` フォルダを開きます
- `初回セットアップ.bat` を「右クリック → 管理者として実行」します
- 画面の案内に従って Excel Add-in と Word Add-in のセットアップを完了します
- セットアップ完了後に Excel と Word を再起動します
- `案件情報System.exe` をダブルクリックして起動します

## 3. 初回セットアップで行われること
- 配布用証明書を登録します
- Excel Add-in のセットアップ画面を開きます
- Word Add-in のセットアップ画面を開きます
- 初回利用に必要な準備をまとめて行います

## 4. 先に実行するもの
- 最初に実行するのは `初回セットアップ.bat` です
- `案件情報System.exe` は、初回セットアップ完了後に実行します

## 5. やってはいけないこと
- ZIP を展開せずにそのまま実行しない
- ZIP の中から一部のファイルだけ取り出して使わない
- `.vsto` を直接ダブルクリックして個別に作業しない
- PowerShell を開いて手動セットアップしない
- `Addins` フォルダを削除しない
- `雛形` フォルダを削除しない
- `logs` フォルダを削除しない
- `案件情報System_Kernel.xlsx` と `案件情報System_Base.xlsx` の名前を変えない
- 古いフォルダに新しい版を上書きしない

## 6. 更新時の注意
- 新しい版を受け取ったら、今のフォルダへ上書きせずに新しく展開します
- 新しい版でも、最初は `初回セットアップ.bat` から始めます
- 切り替え方法の追加案内がある場合は、配布元の案内を優先します

## 7. 配布物に含まれるもの
- `案件情報System.exe`
- `案件情報System_Kernel.xlsx`
- `案件情報System_Base.xlsx`
- `利用開始ガイド.pdf`
- `初回セットアップ.bat`
- `CaseInfoSystem.Internal.cer`
- `Addins` フォルダ
- `雛形` フォルダ
- `logs` フォルダ

## 8. 困ったとき
- `初回セットアップ.bat` が実行できない
- セットアップ画面が途中で止まる
- `案件情報System.exe` が起動できない
- フォルダの中身が足りない

- この場合は、ZIP やフォルダの中身を自分で直さず、そのまま配布元へ連絡してください
- 連絡時は、どの手順で止まったかを伝えてください
